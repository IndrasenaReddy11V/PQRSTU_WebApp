About PQRSTU

PQRSTU is a mobile-first web app for secure, proxy-proof attendance at TERI SAS. It uses dynamic QR codes, GPS geofencing, and real-time updates so students can mark attendance only when present. If a student faces an issue, they can instantly Raise Token for the faculty to accept or deny.
